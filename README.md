# Test Práctico - Frontend

## Features

- Caja de búsqueda
- Resultados de la búsqueda
- Detalle del producto

## Tech
- [React js ] - Vs 18.2.0
- [typescript] - Vs 4.9.4



## Instalacion

Se require [Node.js](https://nodejs.org/) v14+.

```sh
npm i 
```
## Ejecución
Para ejecutar el proyecto
```sh
npm run start
```
## Ejecución de tests
```sh
npm run test or npm run test:coverage
```
